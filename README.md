# 🧠 Python Quiz Web App

A modern and responsive Quiz Web Application built using **Flask**, **HTML**, **CSS**, and **JavaScript**.  
This project allows users to test their programming knowledge through interactive multiple-choice questions with timer-based gameplay and score tracking.

---

# 🚀 Features

✅ Interactive Multiple Choice Quiz  
✅ Timer-Based Questions  
✅ Score Calculation System  
✅ Progress Bar Indicator  
✅ Restart Quiz Functionality  
✅ Responsive Modern UI  
✅ Animated Buttons & Cards  
✅ Glassmorphism Design  
✅ Mobile Friendly Layout  
✅ Performance Result Messages

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/python-quiz-web-app.git
```

---

## 2️⃣ Open Project Folder

```bash
cd python-quiz-web-app
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Create Database

```bash
python database/init_db.py
```

---

## 7️⃣ Run Flask Application

```bash
python app.py
```

---

## 8️⃣ Open In Browser

```plaintext
http://127.0.0.1:5000
```
